- data(){
	dataOneL {} as (Interface or Type)

Просто добавляем ключевое слово as - и делаем type assertion

ПРИ этом на props такая схемка не сработает!

ТУТ вступает понятие GENERIC - и на примере Типизированная функции автор подводит к тому, что 
- если функция принимает в себя какой-то определенный тип, то ее сложно переиспользовать
- поэтому появляются Generic, которые могут принимать в себя различные типы

EventItem - импортируем из типов или прописываем на уровне компонента
PropType импортируем из Vue на уровне компонента

required: true

То есть мы просто указываем глобальный тип Object а потом конкретно пишем, если есть интерфейс или тип
