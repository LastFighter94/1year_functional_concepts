первым делом для работы с TS нам нужно установить TS compiler

npm i -g typescript (делаем это глобально)
################
потом проверяем версию командой

tsc -v
################
чтобы скомпилировать .ts into .js
надо ввести команду tsc name_of_ts_file.ts
после чего в этой же директории возникнет файл js
(команда tsc будет компилировать все .ts файлы найденные в проекте)
(эта команда не игнорирует ts config - однако данная команда может бежать повсюду, что не есть хорошо)

изначальная компиляция идет до ES5
################

tsc --watch name_of_ts_file.ts
будет автоматически отслеживать изменения
################

tsc --init 
инициализация файла конфигурации
################

tsc --watch --target es6 index.ts
команда --target es6 в совокупности с другими 
или отдельно делает так чтобы ts компилировался в es6 версию
################

в конфиге мы можем указать откуда берем ts-files и куда компилируем их в js

"rootDir": "./src" (откуда берем)
"outDir": "./dist" (куда компилируем)

также полезная конфигурация - убирать комментарии при компиляции в js
"removeComments": true

"noImplicitAny": false (здесь ставим false если хотим чтобы был разрешен неявный any, например
при создании функции и "забывании" указания типа на ее аргументы)
