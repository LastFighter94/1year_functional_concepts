The key difference between the Dockerfile and docker-compose is that 

the Dockerfile describes how to build Docker images, - можно описать лишь один образ, который вы подключаете,
А также можно описать настройки этого образа

while docker-compose is used to run Docker containers.
- можно описать множество различных образов, и настройки ко всем этим образам
