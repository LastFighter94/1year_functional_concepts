медиа запросы с указанием определенных условий, позволяют нам выводить те или иные стили
условия эти можно легко комбинировать 

таким образом мы можем определить, например, ориентацию экрана, его размер, цветной он, либо
монохромный и, в соответствии с этими условиями вывести нужные нам стили

на практике все выглядит следующим образом

###################

// первая группа условий - тип устройства
// all - является значением по умолчанию, подходит для всех типов устройств
// print - применяется для документов предварительного просмотра печати
// screen - для цветных мониторов компьютера
// speech - для различных синтезаторов речи 

@media all {

}

!!! медиа запрос с указанием только лишь типа устройства не является break-point-ом

###################

в отличии от указания значений с другой группы, характеристики устройств

// orientation отвечает за положение устройства (портретная либо альбомная)
// orientation:landscape - альбомная (телефон в горизонтальном положении)

@media (orientation:portrait) {

}

###################

// resolution - разрешение (количество точек на дюйм, плотность пикселя)
// можем применить те или иные стили когда наш сайт откроется на экране с подобным разрешением экрана

@media (resolution: 300dpi) {

}

также можно указывать max и min-resolution (максимальное число пикселей на дюйм либо минимальное)
@media (max-resolution: 300dpi) {

}

###################

черно-белый экран - в таком случае break-point сработает

@media (monochrome) {

}

###################

классические break-points

@media (max-width: 767px){
	... (в том случае когда ширина экрана меньше 767px)
}

@media (max-height: 700px){
	... (в том случае когда высота меньше 700px)
}

@media (min-width: 767px){
	... (в том случае когда ширина экрана больше 767px)
}

@media (min-height: 700px){
	... (в том случае когда больше ме700px)
}

###################

УСЛОВИЯ медиа запросов можно комбинировать с помощью логических операторов 

@media screen and (max-width:767px) {

}

@media screen or (max-width:767px) {

}

@media screen and (max-width:767px), screen and (max-height: 700px) {

}

@media screen and (max-width:767px) or screen and (max-height: 700px) {

}

!!!

Медиа запросы можно применять не только в файлах стилей, а например в html при подключении внешнего файла стилей
отдельно, а также импортом в css файл
