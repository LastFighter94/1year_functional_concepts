http - протокол для передачи данных по сети
http запрос - сверху вниз
Строка запроса (указывает метод, URL адрес и версию протокола http)
Заголовки - headers (параметры и прочая информация)
(Пустая строка в качестве разделителя - представляет собой символ перевода строки)
Тело запроса - Entity Body (сами данные, которые передаются в запросе - необязательный параметр)

Get - получения
Post - создание и редактирование (можно удалять данные тоже, но для лучшей семантики эти запросы лучше разделять)
Put - для обновления
Delete - удаление

Web socket - протокол для взаимодействия в реальном времени: realtime взаимодействие.

REST API - один из самых популярных архитектурных подходов к взаимодействию с API (клиент-сервер)
Чаще всего операции CRUD - при обращении к одному и тому же URL с разными HTTP методами
