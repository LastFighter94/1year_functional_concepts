1) computed свойства - вычисляемые, кэшируемый свойства: функции, которые возвращают результат каких-то вычислений

var vm = new Vue({
  el: '#example',
  data: {
    message: 'Привет'
  },
  computed: {
    // геттер вычисляемого значения
    reversedMessage: function () {
      // `this` указывает на экземпляр vm
      return this.message.split('').reverse().join('')
    }
  },
  // кэшируется
  // и обновляется только при изменении message в данном примере
  methods: {
    reverseMessageMethod: function () {
      return this.message.split('').reverse().join('')
    }
  }
  // не кэшируется и будет запускаться при ререндеринге компонента
  // представьте что будет если операция будет более "дорогой"?
})

2) отслеживаемые свойства

объявляются в поле watch и являются функцией, которую необходимо назвать также как и переменную, за которой необходимо следить: будет отрабатывать всякий раз, когда переменная меняет свое значение

	watch: {
		value: (newValue) => {
			console.log('Переменная value отслеживается')
		}
	}

3) как отследить изменение поля объекта

	watch: {
		"obj.field": (newValue) => {
			console.log('Переменная value отслеживается')
		}
	}

4) v-show, v-if отличия

V-if демонтирует элемент из ДОМ, а v-show просто прячет его display: none
