untangle позволяет взять xml file

положить его вот таким образом
xml = untangle.parse(xml_file)
где xml_file = 'xml_для_отправки\\ГКМ.xml'

и далее, обращаться к его схеме через точку

xml.xml.паспорт_гкм[0]
xml.xml.паспорт_гкм[1]

xml.xml.паспорт_гкм[0].геометрия ... и так далее

функция arm.xml.value

passport = xml.xml.паспорт_гкм = [паспорт1, паспорт2, паспорт3, и так далее]
к примеру в xml.xml.паспорт_гкм[1] есть участки недр, а там есть три "участок_недр"

uchs = arm.xml.value(xml.xml.паспорт_гкм[1], 'участки_недр.участок_недр', []) выведет список из трех участков_недр

если в участки_недр пусто, вывелет []
